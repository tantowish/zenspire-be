# ZENIFY

Zenify is a mental health app

## Getting started

**Clone**

```
git clone https://github.com/tantowish/zenify-be.git
cd zenify-be
```

**Setup Environment**

```
cp .env.example .env
```

**Install local dependencies**

```
npm install
```

**Migrate the database**

```
npx migrate dev
```

**Running Service**

```
nodemon run
```
